# Módulo 3: Inteligencia Artificial (Cerebro Gemini)

Lógica central de procesamiento de lenguaje natural y generación de soluciones personalizadas.

### Componentes Clave:
1.  **geminiProxy.ts**: Adaptador para la API de Google Gemini 1.5 Pro con manejo de errores y resiliencia.
2.  **services/aiService.ts**: Abstracción para llamadas de alto nivel (explicaciones, prompts).

### Cómo solicitarlo en un nuevo proyecto:
> "Antigravity, integra el servicio proxy de Gemini de Biotraining para procesar prompts estructurados y manejo de API Keys."
