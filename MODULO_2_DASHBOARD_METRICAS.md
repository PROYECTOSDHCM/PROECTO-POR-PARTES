# Módulo 2: Dashboard y Métricas (Interfaz de Cliente)

El panel de control diseñado para mostrar el ROI y la eficiencia ganada por los agentes de IA.

### Componentes Clave:
1.  **Dashboard.tsx**: Orquestador del panel privado.
2.  **RoiChart.tsx**: Visualización de datos mediante Recharts.
3.  **constants.tsx**: Diccionario de datos maestros para todas las industrias.
